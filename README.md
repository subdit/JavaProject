# Java Project: Email Application 
This Email application is for the java developer who already understand core java and need to practice by using Class and method knowledge.
## The Email application layout. 
   The application should do the following action
	1. To Generate an email with the following Syntax: firstname.lastname@department.company.com
	2. To Determine the department (Sales, development, accounting) if none leave blank
	3. To Generate a random String for a password
  4. To set method to change the password, set the mailbox capacity, and define an alternate email address
  5. To get methods to display the name, email and mailbox capacity
